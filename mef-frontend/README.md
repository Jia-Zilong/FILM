# MEF Frontend

Vue 3 + Vite client for the IF-FILM multi-exposure image fusion demo.

## Setup

```bash
npm install
copy .env.example .env
npm run dev
```

The default API endpoint is `http://127.0.0.1:8000`. Override it with:

```env
VITE_API_BASE_URL=http://127.0.0.1:8000
```

## Scripts

```bash
npm run build
npm run preview
```
